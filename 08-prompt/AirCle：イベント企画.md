【タスク抽出ルール（最重要・厳守）】
============================================================
● **Kanban追記用タスク抽出では、いち（市岡）がするもののみを抽出すること。**
● **議事録の「アクションアイテム（タスク）」セクションには、全てのタスク（いちがするものも、他の人がするものも）を記載するが、Kanban追記用タスク抽出では、いち（市岡）がするもののみを抽出すること。**
● **タスクは3-4つに絞り、最も重要なタスクのみを抽出すること。**
● **必ず含める固定タスク（必須・いちがするもののみ）：**
  1. **主催者側タスク**：企画書提出のリマインド
     - 「[イベント名]イベントの企画書提出リマインド」という形式で記載
     - 期限は企画書提出日の3日前
     - 例：「Difyイベントの企画書提出リマインド（担当：いち／期限：企画書提出日の3日前／優先度：High）」
  
  2. **主催者側タスク**：イベントの告知
     - 議事録タイトルからイベント名を抽出し、「[イベント名]イベントの告知をする」という形式で記載
     - 例：「Difyイベントの告知をする（Peatix・X）（担当：いち／期限：イベント日の1週間前／優先度：High）」

● **その他のタスク（いちがするもののみ）**
  - 上記2つの固定タスクに加えて、議事録の内容から重要なタスク（いちがするもののみ）があれば1-2つ追加（最大4つまで）
  - **まとめられるタスクは必ずまとめる**こと。
    例：「動画編集」と「ノート作成」→「イベントアーカイブの準備（動画編集・ノート作成）」

● **注意：登壇者側タスク（相手がやるべきタスク）は、議事録の「アクションアイテム（タスク）」セクションには記載するが、Kanban追記用タスク抽出には含めないこと。**

● タスクはすべて "動詞 + 目的" の明快な文章にする。
● **タスクは必ず以下の形式で生成すること：**
  ```
  - [ ] #[議事録のタイトル] タスク内容（担当：◯◯／期限：YYYY-MM-DD／優先度：High・Medium・Low）
  ```
  例：議事録タイトルが `2025-11-15_AirCle×Difyイベント企画` の場合
  → `- [ ] #AirCle×Difyイベント企画 タスク内容（担当：◯◯／期限：YYYY-MM-DD／優先度：High・Medium・Low）`
● 期限の形式は必ず `YYYY-MM-DD` 形式（例：2025-11-20）
● 優先度は High / Medium / Low のいずれか
● 締切が明記されていないタスクは "推奨期限（推定）" を付けて構わない。
● 担当者が不明な場合 → 「担当：いち（暫定）」
● 期限不明の場合 → 「期限：次回MTG前（推奨）」と設定し、日付は `date +%Y-%m-%d` で取得した日付から7日後を設定
● **イベント日の特定方法**
  - 議事録の「決定事項」セクションからイベント日を確認
  - イベント日が不明な場合は、文字起こしから推測
  - イベント日が確定していない場合は、暫定的に「次回MTGで確定」と記載

【Kanbanファイルへの自動追記手順（必須実行）】
============================================================
**重要：議事録作成後、必ず以下の手順を実行すること**

1. **議事録タイトルの取得**
   - 生成した議事録のタイトル（`# [日付]_[議事録の種類]`）を取得
   - 例：`# 2025-11-15_AirCle×Difyイベント企画` → プロジェクト名は `AirCle×Difyイベント企画`

2. **Kanbanファイルの読み込み**
   - `read_file` で以下のファイルを読み込む：
     `/Users/ichiokanaoto/Downloads/Ichioka Obsidian/07-To Do/Project_Kanban.md.md`

3. **タスクの追記**
   - 抽出したタスクを、Kanbanファイルの「## AirCle」セクションの下に追記する
   - 追記する際は、既存のタスクの下に追加する
   - タスク形式は必ず `- [ ] #[議事録のタイトル] タスク内容（担当：◯◯／期限：YYYY-MM-DD／優先度：High・Medium・Low）` とする
   - 議事録タイトルから日付部分（`YYYY-MM-DD_`）を除いた部分をプロジェクト名として使用
   - 例：`# 2025-11-15_AirCle×Difyイベント企画` → `#AirCle×Difyイベント企画`

4. **ファイルの保存**
   - `search_replace` または `write` を使用して、追記した内容をKanbanファイルに保存する

5. **Kanbanファイル内のタスクを期限順に並び替える（必須実行）**
   - Kanbanファイル内の「## AirCle」セクションのタスクを、期限（YYYY-MM-DD）の昇順で並び替える
   - 期限が早いタスクが上に来るようにする
   - `search_replace` または `write` を使用して、並び替えた内容をKanbanファイルに保存する

6. **確認**
   - 必要に応じて `read_file` で追記内容と並び順を確認する

**この手順を実行しないと、議事録作成タスクは完了していない扱いとみなします。**
