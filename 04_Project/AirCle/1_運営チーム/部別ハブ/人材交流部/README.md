## ミッション
「Aircleに入ってよかった」を全員に感じてもらう。Slackのあたたかさ、週次交流会、説明会、面談→配属の一連フローを設計・運営する。

## KPI
1. 週次リアクションランキングTop3を人材交流部が独占  
2. 交流会/説明会: 毎月 第1・3週=交流会 / 第2・4週=説明会  
3. 面談制: フォーム返信者100%に1on1実施 / 合格率70%  
4. 新規運営配属: 月3名以上

## アクション動線
1. **Slackウォーミング**: 自己紹介/雑談/質問投稿への即レス。  
2. **イベント設計**: 交流会テーマ・説明会資料・担当割を週次定例で決定。  
3. **面談運用**: 入会フォーム確認→担当割り当て→1週間以内に実施→議事録リンク共有。  
4. **配属&フォロー**: 面談結果を`部一覧_役割分担.xlsx`とSlackロールに反映。
