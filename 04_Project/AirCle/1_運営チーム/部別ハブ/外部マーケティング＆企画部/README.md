- **リーダー**: 大山竜輝（マーケ&デザイン統括）

## ミッション
1. Peatix / note / X / Slack / メルマガの導線を一元管理し、イベントごとにCVRを可視化する。  
2. 告知文・ビジュアル・アーカイブ特典をテンプレ化し、講師の魅力を最大化する。  
3. 営業部・企画運営部から上がる案件を「売れる形」に磨き上げる。  

## KPI
- X投稿: 1日1本（固定ツイ+引用含む）  
- Peatix CVR: 45%以上 / 失注理由の記録100%  
- アーカイブ+note LTV: イベント価格の1.5倍  
- 動画編集リードタイム: 公開から48h以内に納品

## 運用フロー
1. **Brief受領**: 営業部から講師情報とゴールを受け取り、`マーケbrief`テンプレに入力。  
2. **素材制作**: Kanon/デザインチームが1stビジュアル→部内でチェック→Peatix掲載。  
3. **配信計画**: X投稿（ティザー/告知/リマインド/当日/お礼）とメルマガ/LINE/QCを紐付け。  
4. **計測**: 申込数・視聴数・アーカイブ販売・アンケートを記録し、次の企画にフィードバック。
