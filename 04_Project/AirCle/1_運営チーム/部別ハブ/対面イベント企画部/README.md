- 参加者数: 200名 / 回、NPS 60  
- 協賛獲得: 2社以上 / イベント

1. **企画書作成**: 目的・ターゲット・提供価値をドキュメント化（Export参照）。

4. **クロージング**: 参加者フォローとオンラインコミュニティ誘導。
