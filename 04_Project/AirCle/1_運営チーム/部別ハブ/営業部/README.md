# 営業部ハブ
- **リーダー**: 大前綾香 / **メンバー**: はな / 市岡直人（大型案件同席）

## ミッション
AIインフルエンサー・法人との共催・協業を獲得し、Aircleの影響力と売上を拡大する。

## KPI
1. 週5件の新規営業リストを全員が提出。  
2. DM送信は定例終了後〜当日23:59までに完了。  
3. 反応率・次アクションを週次定例で共有し、PDCAを回す。  
4. 成約数: 月2件以上 / 協業パイプライン: 常時8件。  

## 営業フロー
1. **リサーチ**: 候補に関する最新の投稿・ニーズを調査し、個別の価値仮説を設計。  
2. **DM発信**: 「あなたのためにここまで考えました」感を重視。Slackでテンプレ承認後に送信。  
3. **資料/提案**: Notionページへ転記して履歴化。ロゴ・実績・特典を相手仕様にカスタム。  
4. **面談**: フォロワー1万人以上などの大型案件は市岡が同席。ヒアリング→共催案→契約。
