# 会議テンプレート集

## 1. 全社会議（火曜 08:00-09:00）
```
# YYYY-MM-DD 全社会議
0. KPI Snapshot（1分）
1. Opening（今週の目的 / 先週ハイライト）
2. 進捗アップデート（各部30秒）
3. トピックディスカッション（決定が必要な論点のみ）
4. 決定事項まとめ（誰が / いつまでに）
5. アクションアイテム（チェックボックス形式）
6. 後処理チェック（ToDo反映 / Slack共有 / 司会次回設定）
```
- 参考: `0_全体戦略/運営定例_議事録/テンプレート.md`

## 2. 部署定例（週1推奨）
```
# YYYY-MM-DD <部門名> 定例
参加者:
アジェンダ:
1. KPI/数値チェック
2. 進行中タスク（ToDoボード参照）
3. ブロッカー / サポート依頼
4. 人員・リソース確認
決定事項:
アクションアイテム:
次回定例: 日付 / 司会 / 記録係
```
- 定例終了後、`部別ハブ/<部門>/ToDo_ボード.md`と`タスク管理/ToDo_ボード.md`の同期を忘れない。

```
- [ ] リハーサル完了

```

## AI議事録作成の流れ
1. `prompt/AirCle議事録作成プロンプト.md`を開き、会議種別に応じた保存先を設定。  
2. 会議中は最低1名が手動メモを取り、AI議事録との差分を補正。  
3. `保存先`ルール通りに`0_全体戦略`または`部別ハブ`配下へ保存。  

テンプレをObsidianのテンプレート機能かHotkeyに登録すると、誰でも同じ品質で議事録を残せます。
