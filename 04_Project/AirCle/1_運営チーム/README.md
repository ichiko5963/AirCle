# 1_運営チーム ハブ
このディレクトリは運営メンバーが「瞬時に状況を把握し、すぐ動ける」ことに特化したハブです。主要コンテンツは以下の通り。

| パス | 役割 |
| --- | --- |
| `部一覧_役割分担.xlsx` | 全メンバーの責務・Slack連絡先の最新定義。採用・アサイン時はここを起点にする。 |
| `タスク管理/ToDo_ボード.md` | 全社タスクのシングルソース。定例アクション→部別ボードへ流し込む。 |
| `タスク管理/完了済み_アーカイブ.md` | 終了タスクの保管場所。メモやリンクを残し、再現性を担保。 |
| `スケジュール管理.md` | Googleカレンダー連携リンクと、今後8週間のマイルストーン。 |
| `会議テンプレート.md` | 全社会議・部定例・プロジェクト会議のテンプレを収録。 |
| `Slack運用ルール.md` | リアクション/コメント/タグ運用のガイド。新メンバーへの共有必須。 |
| `prompt/AirCle議事録作成プロンプト.md` | AIで議事録を作る際の必須プロンプト。保存先もここで指定。 |
| `部別ハブ/<部門名>/` | 各部のREADME / ToDo / 定例議事録 / テンプレ類を格納。 |

フォルダ構成に沿って運営ドキュメントを整理すると、ToDo↔議事録↔テンプレが自動的にリンクするようになっています。
