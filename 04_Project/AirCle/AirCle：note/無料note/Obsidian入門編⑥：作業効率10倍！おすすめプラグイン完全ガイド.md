## プラグインとは：Obsidianの「拡張機能」

まず、プラグインとは何かを説明します。

**Obsidianは、シンプルで美しい。でも、デフォルトの機能だけでは限界があります。**

プラグインは、**Obsidianに機能を追加する「拡張パーツ」**です。

### プラグインの種類

Obsidianには2種類のプラグインがあります：

**1. コアプラグイン（標準機能）**

- Obsidianに最初から含まれている
- 設定でオン・オフできる
- 例：デイリーノート、グラフビュー、検索

**2. コミュニティプラグイン**

- 世界中の開発者が作った拡張機能
- 無料で使えるものがほとんど
- 例：Tasks、Kanban、AI連携

### プラグインのインストール方法（1分）

**初回のみ、一度だけ設定**：

1. Obsidianを開く
2. 左下の**歯車マーク（設定）**をクリック
3. 左サイドバーの「コミュニティプラグイン」をクリック
4. 「コミュニティプラグインを有効化」ボタンをクリック（初回のみ）
5. 「閲覧」ボタンをクリック

これで、プラグインの検索・インストールが可能になります。

**プラグインをインストールする手順**：

1. 検索窓にプラグイン名を入力
2. 見つかったら「インストール」をクリック
3. 「有効化」をクリック
4. 必要に応じて設定（歯車マーク）を調整

**完了！**

では、カテゴリ別におすすめプラグインを紹介します。

## カテゴリ1：起点づくり／ナビゲーション

毎日Obsidianを開いた時、「今日は何をしよう？」と迷いますか？

この3つのプラグインで、迷いがなくなります。

### Homepage：起動時に「ダッシュボード」を表示

**役割**：Obsidian起動時に、任意のノートやキャンバスを自動表示。タブの扱いや表示モードの指定も可能。

**なぜ重要か**：

毎朝、Obsidianを開いて「今日は何をしよう？」と迷う時間、もったいないですよね。

Homepageを設定すれば、起動時に自動で今日のタスクや重要リンクが表示されます。迷いがゼロになります。

**導入方法**：

1. 「コミュニティプラグイン」で「Homepage」を検索
2. インストール＆有効化
3. 設定で「Homepage note」を指定（例：「今日のタスク」というメモ）

**使い方**：

今日のタスク、今週の目標、重要リンクなどをまとめた「ダッシュボード」ノートを作ります。

```markdown
# 今日のタスク

## やること
- [ ] [[企画案_新商品]]を完成させる
- [ ] [[マーケティング記事]]を読む
- [ ] [[田中さんとの打ち合わせ]]に備える

## 今週の目標
- [[Zettelkasten]]の手法をマスターする
- 10個のメモを作る
```

これをHomepageに設定すれば、起動時に自動で開きます。

**導入TIP**：週次レビューや「今日のタスク」ノートを指定して、認知負荷をゼロに。

### Outline（コアプラグイン）：長文の構造を可視化

**役割**：アクティブノートの見出しをサイドバーにツリー表示。クリックでジャンプ、ドラッグでセクション並べ替えが可能。

**なぜ重要か**：

1000文字、2000文字のメモになると、「あの部分、どこだっけ？」と探すのに時間がかかります。

Outlineがあれば、メモの構造が一目瞭然。クリック一つで該当箇所にジャンプできます。

**設定方法**：

1. 設定（歯車マーク）→「コアプラグイン」
2. 「Outline」をオンにする
3. 右サイドバーに表示されます

**使い方**：

長文のメモを書いている時、右サイドバーのOutlineを見ると：

```text
# マーケティング戦略
  ## 4P理論
    ### プロダクト
    ### プライス
  ## 実践例
```

クリックで該当箇所にジャンプ。ドラッグでセクションを並べ替えも可能です。

**導入TIP**：分割ビューと併用し、左：執筆、右：アウトラインのワークスペースを固定。

### Quick Switcher（コアプラグイン）：キーボードで高速移動

**役割**：⌘/Ctrl+O でノート名をインクリメンタル検索→即オープン。キーボード中心の高速移動軸。

**なぜ重要か**：

マウスでクリックしてノートを探すのは遅い。キーボードで一瞬で移動できれば、作業効率が10倍になります。

**使い方**：

1. ⌘/Ctrl+O を押す（Macは⌘、WindowsはCtrl）
2. ノート名の一部をタイプ
3. Enterでオープン

例：「マーケ」とタイプすると、「マーケティング戦略.md」「マーケティング4P理論.md」が候補に出ます。

**補強**：高度検索が必要なら「Another Quick Switcher」等で拡張も可能。

### Command Palette（コアプラグイン）：全機能に即アクセス

**役割**：⌘/Ctrl+P から全コマンドに即アクセス。よく使うコマンドのピン留めで更に短縮。

**使い方**：

⌘/Ctrl+P を押すと、Obsidianのすべての機能にアクセスできます。

- 「新しいノートを作成」
- 「グラフビューを開く」
- 「検索」
- 「設定」

キーボードだけで、すべて操作可能です。

## カテゴリ2：情報取り込み／埋め込み

前回の記事で、Web ClipperとKindle Highlightsを紹介しました。

でも、それだけでは物足りない。もっと強力な取り込み機能があります。

### Obsidian Web Clipper（公式拡張）：Web記事の完全保存

**役割**：Webページの保存・ハイライトをローカルMarkdownで長期保全。プライバシー前提の設計。

**なぜ重要か**：

「後で読もう」と思ってブックマークした記事、実際に読んだことありますか？

Web Clipperなら、**記事の内容をObsidianに保存**できます。オフラインでも読めます。プライバシーも守られます。

**インストール方法**（前回の記事で詳細解説）：

1. Chrome Web Storeで「Obsidian Web Clipper」を検索
2. 「Chromeに追加」をクリック
3. Obsidianと連携

**導入TIP**：テンプレにsource, author, retrieved_at等を自動挿入すると、後で出典を確認できます。

### Open Gate：WebサイトをObsidian内に直接埋め込み

**役割**：あらゆるWebサイトをObsidian内に直接埋め込み（専用ビュー／ノート内インライン）。プロファイル共有やCSS/JS注入も可。

**なぜ重要か**：

Notion、Googleカレンダー、ChatGPT、Wikiなど、「このサイトを参照しながらメモを書きたい」と思ったことはありませんか？

Open Gateがあれば、**WebサイトをObsidian内に直接表示**できます。ブラウザを切り替える必要がありません。

**ユースケース**：

- Notion・Googleカレンダー・ChatGPT・Wiki等の"ライブ参照"
- YouTube講義とノートの同時処理
- 資料を見ながら、メモを書く

**インストール方法**：

1. 「コミュニティプラグイン」で「Open Gate」を検索
2. インストール＆有効化

**使い方**：

```markdown
![[https://example.com]]
```

これだけで、Webサイトがノート内に埋め込まれます。

### Convert URL to Preview (iframe)：動画学習に最適

**役割**：URLを右クリック→iframeプレビューへ即変換（YouTubeのタイムスタンプも保持）。動画学習ノートで強力。

**なぜ重要か**：

YouTubeで学ぶことが多い現代。動画の内容をメモしながら、該当箇所を再生したい、と思ったことはありませんか？

このプラグインがあれば、**YouTubeをObsidian内で再生しながら、メモが書けます。**

**インストール方法**：

1. 「コミュニティプラグイン」で「Convert URL to Preview」を検索
2. インストール＆有効化

**使い方**：

ノート内にURLを貼り付け、右クリック→「Convert to iframe preview」を選択。動画が埋め込まれます。

## カテゴリ3：AIアシスト

AI時代、Obsidianも進化しています。

あなたのメモをAIが理解し、執筆をサポートする。そんな未来が、もう来ています。

### Copilot for Obsidian：あなたの知的執事

**役割**：ボールト横断検索×チャット、要約・下書き生成、YouTubeやWebの文脈取り込み等のAI執事。データ主権を維持しつつ高度なエージェント機能を提供。

**なぜ重要か**：

「過去にメモしたあのアイデア、どこだっけ？」「このメモの要点を教えて」「企画書の下書きを作って」

AIに聞けば、あなたのObsidianから情報を探し出し、要約し、執筆をサポートしてくれます。

**主な機能**：
- ボールト全体の検索
- メモの要約
- 下書き生成
- YouTubeやWebの文脈取り込み

**インストール方法**：

1. 「コミュニティプラグイン」で「Copilot for Obsidian」を検索
2. インストール＆有効化
3. 設定でAPIキー（Gemini等）を入力

**補足**：Gemini等で利用する実践記事も豊富（導入・APIキー設定手順の参考に）。

**使い方**：

Copilotのチャット画面で：
「私のObsidianにあるマーケティング関連のメモを要約して」

→ 関連するメモが検索され、要約が生成されます。

「[[企画案_新商品]]を元に、プレゼン資料の下書きを作って」

→ 企画案のメモを読み込み、下書きを生成します。

**これがあれば、ObsidianがAI執事のように働いてくれます。**

## カテゴリ4：タスク／カレンダー

Obsidianは知識管理ツールですが、タスク管理もできます。

すべてを一箇所に集約することで、**知識と行動が繋がります**。

### Tasks：ボールト全体のタスクを集約

**役割**：ボールト横断でチェックリストを集計。期日・繰り返し・完了日・フィルタ構文を持ち、どこからでもトグル更新可能。

**なぜ重要か**：

- メモ内に「- [ ] 明日やること」と書く
- デイリーノートに「- [ ] 週末に読む本」
- 企画案に「- [ ] データ収集」

すべてのタスクがバラバラ。管理できません。

Tasksがあれば、**すべてのタスクが一箇所に集約**されます。

**インストール方法**：

1. 「コミュニティプラグイン」で「Tasks」を検索
2. インストール＆有効化

**使い方**：

メモ内にタスクを書きます：

```markdown
- [ ] マーケティング記事を読む 📅 2025-01-15
- [ ] 企画案を完成させる 🔁 every week
```

Tasksプラグインが自動で集計します。

**運用TIP**：#task等のグローバルフィルタを決めてビューのノイズを排除。

### Calendar（コアプラグイン）+ Daily Notes：日次記録を習慣に

**役割**：日付ノートをカレンダーUIで作成・移動。日次記録・振り返りを"1クリック"に。

**設定方法**（前回の記事で詳細解説）：

1. 設定→「コアプラグイン」→「デイリーノート」をオン
2. 「カレンダー」もオン
3. 左サイドバーにカレンダーアイコンが表示されます

**使い方**：

カレンダーアイコンをクリック→今日の日付のノートが自動生成されます。

毎日の振り返り、学んだこと、明日やること...すべてを一箇所に記録できます。

**導入TIP**：週次レビューや「今日のタスク」ノートを指定して、認知負荷をゼロに。

## カテゴリ5：可視化／UI

視認性が上がれば、モチベーションも上がります。

### Kanban：アイデアや案件をカード管理

**役割**：Markdown基盤のボードでアイデア／案件をカード管理。進捗やカテゴリ分けに直感的。

**なぜ重要か**：

プロジェクトを管理する時、「やること」「作業中」「完了」で分けたい、と思ったことはありませんか？

Kanbanがあれば、**視覚的に進捗を管理**できます。

**インストール方法**：

1. 「コミュニティプラグイン」で「Kanban」を検索
2. インストール＆有効化

**使い方**：

新しいノートを作り、最初に「---\nkanban-plugin\n---」を書きます。

```markdown
## やること

- [ ] データを収集

## 作業中
- [ ] プレゼン資料を作成
```

カード形式で表示され、ドラッグで移動できます。

**補足**：要件によっては「Projects」系のボードも検討余地あり。

### Iconize：ファイルにアイコンを付与

**役割**：ファイル／フォルダ／見出し等にアイコン付与。視認性と操作モチベーションの両立。

**なぜ重要か**：

「このフォルダ、何だったっけ？」「あのメモ、どれだっけ？」

アイコンがあれば、**一目で分かります**。視認性が上がり、操作が楽しくなります。

**インストール方法**：

1. 「コミュニティプラグイン」で「Iconize」を検索
2. インストール＆有効化

**使い方**：

ファイル名やフォルダ名の前に、絵文字やアイコンを追加するだけです。

```text

📝 デイリーノート
💡 アイデア
📊 プロジェクト
```

### Themes：外観をカスタマイズ

**役割**：外観テーマの切替と配色・UI最適化（集中・可読性・ワークスペースの一貫性向上）。

**なぜ重要か**：

毎日使うツールだから、**見た目が気に入った方が長続きします**。

**使い方**：

1. 設定→「外観」→「テーマ」
2. テーマを選択（ダークモード、ライトモード等）

**導入TIP**：Iconize等と併用すると、さらに見やすくなります。

## カテゴリ6：情報整理／分割ビュー

メモが増えてくると、「整理」が必要になります。

でも、整理の仕方も、Obsidianなら簡単です。

### Note Refactor：長文を分割して整理

**役割**：選択テキストを新規ノートへ抽出し、元ノートにバックリンクを残す。長文の章分け・スライド素材化に有効。

**なぜ重要か**：

1つのメモが5000文字、10000文字になると、「管理が大変...」と感じます。

Note Refactorがあれば、**長文を分割して、新しいメモを作成**できます。元のメモには自動でリンクが残ります。

**インストール方法**：

1. 「コミュニティプラグイン」で「Note Refactor」を検索
2. インストール＆有効化

**使い方**：

1. 分割したい部分を選択
2. 右クリック→「Note Refactor: Extract Selection」
3. 新しいメモが作成され、元のメモにリンクが自動挿入

**活用例**：

- 長い読書メモを章ごとに分割
- 企画案を細かく分割して管理
- スライド用の素材に変換

### Auto Note Mover：自動でフォルダ分け

**役割**：タグ／タイトル等のルールでノートを自動仕分け。クリップ直後の振り分けを省力化。

**なぜ重要か**：

Web Clipperで記事を保存したら、手動で「03_ウェブ」フォルダに移動...面倒です。

Auto Note Moverがあれば、**ルールを設定して自動で移動**できます。

**インストール方法**：

1. 「コミュニティプラグイン」で「Auto Note Mover」を検索
2. インストール＆有効化

**使い方**：

設定でルールを追加します：

- タグ「#web」がついていたら→「03_ウェブ」フォルダへ
- タイトルに「【記事】」が含まれていたら→「03_ウェブ」フォルダへ

**導入TIP**：クリップ直後の振り分けを省力化できます。

### Split Down / Split Right（コア操作）：分割ビューで同時作業

**役割**：タブを上下・左右に分割し、資料×執筆／本文×プレビュー等の同時作業を実現。タブの右クリックから実行。

**なぜ重要か**：

- 資料を見ながら、メモを書きたい
- 本文を書きながら、プレビューを見たい
- 2つのメモを比較したい

分割ビューがあれば、**複数のファイルを同時に見ながら作業**できます。

**使い方**：

1. タブを右クリック
2. 「Split right」（左右分割）または「Split down」（上下分割）を選択
3. 2つのペインで作業可能

**活用例**：
- 左：Web記事、右：メモ（情報を取り込みながら執筆）
- 上：企画案、下：関連データ（比較しながら執筆）
- 左：本文、右：プレビュー（確認しながら執筆）

**補足**：モバイルでの挙動は端末依存の報告あり。

「あの動画、良いこと言ってたけど、何だったっけ？」「時間がないから、文字起こしだけ読めたら...」

YTranscriptがあれば、**YouTubeの字幕を取得して、ノートに変換**できます。タイムスタンプも残るので、該当箇所をクリックして再生できます。

**インストール方法**：

1. 「コミュニティプラグイン」で「YTranscript」を検索
2. インストール＆有効化

**使い方**：

1. YouTubeのURLをノートに貼り付け
2. コマンドパレット（⌘/Ctrl+P）→「YTranscript: Get Transcript」

**実務TIP**：Media Extended等との併用や、整形の正規表現Tipsがコミュニティに蓄積されています。

## プラグイン選びの原則：必要なものだけを

プラグインは便利ですが、**すべてをインストールする必要はありません**。

### プラグイン選びの基本原則

**重要なのは「すべてを入れることではなく、必要なものだけを選ぶこと」です。**

### 選び方の原則

**1. まずはコアプラグインから**

- Outline
- Quick Switcher
- Command Palette
- Calendar + Daily Notes

これだけで、十分強力です。

**2. 自分のワークフローに必要なものだけを**

- Web記事をよく読む → Web Clipper、Open Gate
- YouTubeで学ぶ → YTranscript
- タスク管理したい → Tasks
- プロジェクト管理 → Kanban

**3. 一度に1つずつ導入**

プラグインを一度に10個導入すると、混乱します。

1週間使って、慣れてきたら次を追加。これが成功の鍵です。

## よくある質問：プラグイン編

### Q1: プラグインを入れすぎると、重くなりませんか？

A: はい、可能性があります。でも：

- 必要なものだけを選べば問題なし
- 10個程度なら、ほとんどのPCで快適
- 重くなったら、使わないプラグインを無効化

### Q2: 有料プラグインは必要ですか？

A: 基本的に**無料で十分**です。

- ほとんどのプラグインは無料
- 有料機能があるプラグインも、基本機能は無料
- まずは無料で始めて、必要になったら有料化を検討

### Q3: プラグインが更新されたら、自動で更新されますか？

A: はい、Obsidianが通知します。

- 設定→「コミュニティプラグイン」→更新があると表示
- ワンクリックで更新可能

### Q4: プラグインがバグったらどうすれば？

A: 簡単です。

1. 該当プラグインを無効化
2. Obsidianを再起動
3. プラグインのGitHubやフォーラムで報告
4. または、別のプラグインを試す

### Q5: スマホでもプラグインは使えますか？

A: 部分的に使えます。

- Obsidian Mobileは基本的なプラグインに対応
- ただし、すべてのプラグインが対応しているわけではない
- モバイルでの挙動は端末依存の報告あり
