## 革命1：「毎回ゼロから」が「6割完了から」になる

### 従来の仕事：シーシュポスの呪い

ギリシャ神話に、シーシュポスという王がいます。

彼の罰は：岩を山頂まで押し上げる。頂上に着くと、岩は転がり落ちる。また押し上げる。これを永遠に繰り返す。

**これが、知識労働者の現実です。**

- 企画書を書く→完成→また次の企画書をゼロから
    
- プレゼン資料を作る→発表→また次の資料をゼロから
    
- レポートを書く→提出→また次のレポートをゼロから
    

毎回、同じ岩を押し上げています。

### Obsidianという革命

**ルーマンの言葉**：

> 「私の生産性の秘密は、毎回ゼロからやらないことだ。」

Obsidianを使うと：

- 企画書を書く→その過程で得た知識がメモとして蓄積される
    
- 次の企画書→前回のメモ、データ、アイデアが既にある
    
- 3回目の企画書→過去2回分の知識が複利で効く
    

**あなたは、山の中腹からスタートしています。**

### 具体的に何が起こるか

**Before（ゼロから）**：

```
情報収集：2時間
資料探し：1時間
構成検討：1時間
執筆：2時間
合計：6時間
```

**After（6割完了から）**：

```
Obsidian検索：5分
関連メモを辿る：10分
構成（既存メモの組み合わせ）：15分
執筆：1時間
合計：1.5時間
```

**4倍速。これが毎日起こります。**

半年後、あなたは気づきます。 「私、同僚の4倍の仕事をこなしている」

でも、残業はしていません。 なぜなら、毎回ゼロからやっていないから。

### この革命が起こる理由

**知識の複利効果**

お金の複利は有名です。100万円を年利5%で30年運用すると、432万円になります。

**知識にも複利があります。**

- 1日目：メモ1個
    
- 7日目：メモ7個
    
- 30日目：メモ30個
    
- でも、価値は30倍ではありません
    

なぜか？リンクで繋がっているから。

30個のメモが相互にリンクされていると：

- 組み合わせの数は膨大
    
- 新しい視点が生まれる
    
- 「点」が「面」になる
    

**半年後、あなたの知識ベースは、半年分の価値ではありません。**

指数関数的に成長しています。

## 革命2：「ネタ切れ」から「ネタがありすぎて困る」へ

### なぜネタ切れが起こるのか

1. ネタを思いつく
    
2. すぐ書く（または忘れる）
    
3. 書き終わったら、脳内はゼロにリセット
    
4. 次のネタを、また「思いつく」必要がある
    

**脳は、ネタの「貯蔵庫」ではありません。** 思いついても、忘れます。

### Obsidianという貯蔵庫

**デイリーノートの魔法**

毎日、5分だけ、デイリーノートを書きます。

```
# 2025-01-15

## 今日の気づき
- 朝の通勤で考えたこと
- ランチタイムの会話から
- 夕方に読んだ記事の感想
```

これを半年続けると：

- 180日分の気づき
    
- 少なくとも180個のネタの種
    
- しかも、リンクで繋がっている
    

**ネタを「探す」必要がなくなります。** **ネタは「既にある」からです。**
