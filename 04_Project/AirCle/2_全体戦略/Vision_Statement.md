# AirCle Vision Statement

## 存在意義 / Purpose
Aircleは「AIを武器に“つくり切る”学生・若手を増やす」ために生まれました。従来のインプット型コミュニティではなく、ゼロからプロダクト・講座・受託案件を生み出す実践基地として機能します。Slackに集う150名超のメンバーが、学習→企画→実装→収益化までを1つの動線で駆け抜けられる環境を提供し、参加者の市場価値を半年単位で引き上げます。

## Vision 2030 Snapshot
1. **「作れる」集合体**: 参加メンバーの50%が自走的にサービス/講座/受託案件をローンチできる状態。  
2. **AI人材のハブ**: 国内で最も実戦的な学生AIコミュニティとして、企業や行政から「共創パートナー」として指名される。  
3. **循環型エコシステム**: イベント収益・受託収益を運営/メンバーへ再投資し、挑戦→実績→再挑戦の循環を回し続ける。  
4. **クロスファンクショナル文化**: 部門・専門領域を越えてプロジェクトを結成し、最短距離で価値提供できるフラットな文化を維持する。  

## Core Values
- **Make & Ship**: 「学ぶ > 作る > 届ける」を短サイクルで回し、未完成でもリリースして学ぶ。  
- **Warm but Demanding**: 人材交流部が中心となり、Slackは常にリアクション・歓迎コメントで溢れる。ただし甘さに妥協せず、期日と責任を徹底する。  
- **Data-driven Ownership**: Ryucleなどダッシュボードを通じてリアクション数、参加率、売上を可視化。各リーダーがKPIを持ち、定例で意思決定する。  
- **Give Back the Spotlight**: 登壇者・制作チームが得た知見を有料noteやSlack限定コンテンツで還元し、次の挑戦者を後押しする。  

## Value Proposition
| ステークホルダー | 提供価値 | 主要タッチポイント |
| --- | --- | --- |
| 学生メンバー | 無料ライブ参加、月額アーカイブ視聴、実務レベルの運営/制作経験 | Slack、交流会、1on1、運営プロジェクト |
| 社会人参加者 | 高密度なAI講座・アーカイブ・noteバンドル、即実践のテンプレ | Peatix、YouTube、メールマガ、note |
| 講師/インフルエンサー | リスト拡大＋売上シェア（イベント20%/アーカイブ販売1.5倍設定） | 営業部→外部マーケ導線 |
| パートナー企業 | 共催イベント、受託（構築/運用/動画）による成果創出 | 経営部・営業部による窓口 |

## Strategic Pillars (FY2025)
1. **Event Flywheel**: 11-12月はObsidian×Cursorなど8本の大型講座を集中投入。アーカイブ＋noteでLTVを1.5倍化。  
2. **Community to Crew**: 面談制オンボーディングと交流会/説明会（第1・3週=交流、第2・4週=説明）で“熱量の高い仲間”を特定・配属。  
3. **Productized Services**: Lark構築、動画編集、AIツール内製等のチームを立ち上げ、受託と自社プロダクトの両輪を構築。  
4. **Operational Excellence**: モデレーター品質マニュアル、Slack反応率KPI、部別ToDo/議事録テンプレを標準化し、誰でも即戦力参加できる土台を整える。  

## Success Metrics
- 月次イベント数 ≥ 6本 / 平均参加者 400名 / 満足度4.5以上  
- アクティブSlackメンバー比率 20% → 35%  
- 面談通過者の運営参画率 80%  
- 受託/共同案件 4件（Lark構築2件、動画編集1件、SNS運用1件）  
- アーカイブ+note サブスクリプション 月次¥999 × 300名  

このビジョンが、0→1を量産し続ける「Aircle 2.0」の意思決定基準になります。
