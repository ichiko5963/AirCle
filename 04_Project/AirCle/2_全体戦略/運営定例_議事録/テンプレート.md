# 運営定例テンプレート
> 週次全社会議（火曜 08:00-09:00 JST）用テンプレート。Obsidianの`Daily notes`か当フォルダ配下にコピーして使用する。

```
# YYYY-MM-DD_定例会議事録
日付: YYYY-MM-DD
時間: HH:MM - HH:MM
場所: Zoom / YouTube / 対面（選択）

## 出席者
- 出席:
- 欠席:

## アジェンダ
1. Opening（目的・先週ハイライト）
2. 各部30秒アップデート（企画運営/人材交流/営業/外部マーケ/経営/対面イベント/技術）
3. トピックディスカッション（決めるべき論点のみ）
4. 決定事項まとめ
5. アクションアイテム
6. 次回会議・後処理チェック

## 決定事項
- [ ] 例: Lark構築のPoC開始（担当: TK / 期限: 12/10）

## アクションアイテム
- [ ] タスク｜責任者｜期限

## 議事録（詳細）
- HH:MM 記録者: 要点
- 参考リンク: ...

## 後処理チェック
- [ ] 決定事項を`ToDo_ボード.md`へ反映
- [ ] 部別タスクへ振り分け
- [ ] Slack #aircle-経営部 へ共有
```

### 保存ルール
1. `date +%Y-%m-%d`で日付確認。  
2. 本フォルダに`YYYYMMDD.md`で保存（例: `20251111.md`）。  
3. ToDo/決定事項を`1_運営チーム/タスク管理/ToDo_ボード.md`に反映。  
4. 部別タスクは各部ハブの`ToDo_ボード.md`へリンク。
