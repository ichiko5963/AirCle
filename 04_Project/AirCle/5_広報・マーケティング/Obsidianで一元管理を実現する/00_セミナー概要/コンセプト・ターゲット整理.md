# コンセプト・ターゲット整理

## セミナーのコンセプト
「Obsidian × AI によるドキュメント活用ワークフロー」

### 核心メッセージ
- 情報の一元管理を実現する具体的な方法を学ぶ
- 個人ワークフローとチームワークフローの2軸で運用例を紹介
- 「情報収集 × AI活用」を中心とした、Obsidianを核としたドキュメント管理フローを体系的に解説
- 一次情報をObsidianに自動集約し、AIによる要約・構造化へつなげる
- 外部ドキュメント（Notion、Google Meet）を安全にObsidianへ取り込む仕組み
- AI活用に最適化されたフォルダ構成で、情報の鮮度管理やAI活用時の効率化を実現

## ターゲット層

### プライマリターゲット
- **Obsidianを使っているが、情報の一元管理ができていない人**
  - 情報が散在していて、どこに何があるか分からない
  - 複数のツールを使い分けているが、統合できていない
  - 情報を集めても活用できていない
  - 情報の一元管理の具体的な方法が分からない

- **情報収集はしているが、AI活用までつなげられていない人**
  - 情報を集めるだけで満足してしまっている
  - AIを活用したいが、どう組み込めばいいか分からない
  - 情報を構造化して活用したいが、方法が分からない

### セカンダリターゲット
- **個人ワークフローを効率化したい人**
  - Kindle、RSS、Web Clipperなどで情報を集めているが、整理できていない
  - 一次情報を効率的にObsidianへ取り込みたい
  - AIによる要約・構造化を活用したい

- **チームワークフローを改善したい人**
  - NotionやGoogle Meetの議事録をObsidianで管理したい
  - 複数人のドキュメントを安全に統合したい
  - 読み取り専用で外部ドキュメントを統合したい

### ターシャリターゲット
- **GitHubを活用した自動化に興味がある人**
  - Notion → Obsidian の同期を自動化したい
  - Google Meet → Obsidian の同期を自動化したい
  - RSSやメモの自動取得を実現したい

- **AIルール設定（Cursor）に興味がある人**
  - プロンプトテンプレートを活用したい
  - ノート整形ルールを設定したい
  - カテゴリ分類や情報フィルタリングを自動化したい

## ペインポイント
1. **情報の散在**: 情報が複数のツールに散らばっていて、どこに何があるか分からない
2. **一元管理の難しさ**: 一元管理したいが、具体的な方法が分からない
3. **情報収集だけ**: 情報を集めるだけで、活用までつなげられていない
4. **AI活用の壁**: AIを活用したいが、どう組み込めばいいか分からない
5. **手動での整理**: 手動で情報を整理するのが面倒で続かない
6. **フォルダ構成の迷い**: どのようにフォルダを構成すればいいか分からない
7. **個人とチームの分離**: 個人ワークフローとチームワークフローをどう統合すればいいか分からない
8. **外部ドキュメントの統合**: NotionやGoogle Meetの情報をObsidianで管理したいが、方法が分からない

## ソリューション
1. **個人ワークフロー**: 一次情報をObsidianへ自動集約する方法（Kindle Highlights Importer、Local RSS、Line Touch Ink、Obsidian Web Clipper）
2. **チームワークフロー**: 外部ドキュメントを読み取り専用で統合する仕組み（Notion API × GitHub Actions、Google Meetログの取り込み）
3. **AI活用に最適化されたフォルダ構成**: Read Only、AI Output、Archive、Notesの構造で情報の鮮度管理とAI活用時の効率化を実現
4. **GitHubを活用した自動化**: Notion → Obsidian、Google Meet → Obsidianの同期を自動化
5. **AIルール設定（Cursor）**: プロンプトテンプレート、ノート整形ルール、カテゴリ分類ルール、情報フィルタリング
6. **テンプレート配布**: 個人ワークフロー用「Obsidian保管庫（Vault）テンプレート」を配布

## 期待される成果
- 情報の一元管理を実現できる
- 個人ワークフローとチームワークフローを統合できる
- 一次情報をObsidianへ自動集約できる
- AIによる要約・構造化を活用できる
- 外部ドキュメント（Notion、Google Meet）を安全にObsidianへ取り込める
- AI活用に最適化されたフォルダ構成を理解し、実装できる
- GitHubを活用した自動化の基礎を習得できる
- AIルール設定（Cursor）を活用してワークフローを最適化できる
