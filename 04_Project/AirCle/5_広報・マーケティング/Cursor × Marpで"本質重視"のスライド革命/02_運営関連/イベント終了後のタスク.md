# イベント終了後のタスク

## 即座に実行するタスク

### イベント終了直後
- [ ] 参加者への感謝メッセージ配信
- [ ] 特典配布の案内
- [ ] アンケート依頼（任意）

## 2〜3日以内に実行するタスク

### 動画編集・配布
- [ ] アーカイブ動画の編集（フィラーワードカット）
- [ ] 有料チケット購入者へのアーカイブ動画配布
- [ ] 運営執筆noteの作成・配布

### 特典配布
- [ ] 無料参加者全員への特典配布（Marpテンプレ、スライド生成プロンプト）
- [ ] 配布方法の案内（Peatixメッセージ機能）

## 1週間以内に実行するタスク

### 実績アーカイブ
- [ ] 申込者数・参加者数の記録
- [ ] 参加者アンケートまとめ（実施した場合）
- [ ] 次回改善点の記録

### 分析・報告
- [ ] 分析レポートの作成
- [ ] 効果的だった手法の記録
- [ ] 改善すべき点の明確化

## 継続タスク

### フォローアップ
- [ ] 参加者からの質問への対応
- [ ] コミュニティでのフォローアップ
- [ ] 次回イベントへの誘導
