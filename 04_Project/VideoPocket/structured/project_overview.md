# Project Overview

## プロダクトの基本情報
- 名称：VideoPocket（動画編集AI SaaS／制作自動化OS／クリエイター支援プラットフォーム）
- 開発：ひかる｜動画編集AI (@hikarun_videoai)
  - 動画の民主化を掲げ、TikTok 9,000フォロワー、東大AI研究バックグラウンド
- ミッション：**動画制作の民主化で人類の映像生産性を100倍にする**
  - 誰でも動画制作できる世界へ
  - 編集の作業負荷をAIで肩代わりし、クリエイター時給を劇的に上げる
  - 参入障壁ゼロのノータッチ運用体験を作る

## プロダクト思想（Philosophy）
1. **Zero-Touch Experience**：アップロード即フル自動処理
2. **動画制作のOS化**：編集〜生成〜投稿〜分析までを統合
3. **人間の創造性の最大化**：人は企画・判断・感性だけに集中
4. **映像の民主化**：専門家領域を一般層に開放
5. **AIネイティブ編集者の創出**：次世代クリエイタースキルの標準化

## 価値提供の全体像
- AIによる編集自動化で作業時間を1/10にし、ショート量産を可能にする
- v2以降はコンテンツ生成・切り抜き提案まで自動化し、SNS運用も含めたOS化
- v3では“学習型プロデューサーAI”が各アカウントの勝ち筋を学ぶ
- VideoPocket単体だけでなく、先行教材「動画AIポケット」・Discordコミュニティ・Manus連携などでエコシステム化

## プロダクトカテゴリ別の位置づけ
| カテゴリ | 具体像 |
| --- | --- |
| 動画編集AI SaaS | 字幕・カット・テロップ・誤字修正を完全自動化 |
| 動画制作AIエージェント | 切り抜き案、運用提案、投稿代行をAIが行う |
| 動画自動化OS | Manusやn8nと連携しワークフローをテンプレ化 |
| クリエイター支援PF | ノート教材、ウェビナー、Discordで育成コミュニティ化 |

## 戦略的キーメッセージ
- 「ノータッチで映像量産」「民主化」「AIネイティブ編集者」という文脈で発信
- OpusClipをベンチマークしつつ、より日本市場に刺さるデザインとUXで差別化
- v1/v2を布石に、動画運用エージェント（v3）でサブスク売上を積み上げる
