# Product Requirements

## 機能要件（Functional）
### 基本操作
- 動画アップロードと自動処理キュー
- 進行ステータス／エージェント処理ログの可視化

### 編集自動化
- VADベースの無音カット（Premier以上の自然さ + 事故カット防止）
- 字幕生成（AI文字起こし→整形→SRT書き出し）
- 誤字修正・句読点挿入・フィラー削除
- テロップ要約（重要部分を抽出）

### 生成（コンテンツオートメーション）
- 切り抜き生成（1本→最大10案、TikTok/YT Shorts最適化）
- 要約動画生成（長尺→30秒ハイライト）
- テーマ特化生成：雑学ショート／ニュースショート／スポーツハイライト／AIおばあちゃん動画／商品紹介UGC／アバター広告

### エージェント機能
- 投稿時間最適化とSNS予約投稿
- コメント解析と再編集提案
- アカウント成長/KPI管理ダッシュボード
- 企画案・サムネ案・コピー作成

### 出力
- mp4動画、srt字幕、jsonメタデータ（切り抜き位置・要約テキストなど）

## 非機能要件（Non-Functional）
- 高速処理（UXで待たせない）
- Manus / n8n / Sora系モデルとの堅牢なAPI連携
- 拡張性と安定性を両立しレイテンシ最小化
- スマホ最適化とシンプルな導線

## UX / UI 要求
- 5ステップ以内で完結する直感的UI
- 自動処理の進行状況を常に提示し不安を減らす
- 機能群が多くても迷わないインフォメーションアーキテクチャ
- 動画編集未経験者でも理解できるコピーとレイアウト
- ノータッチを演出するマイクロインタラクション（例：Auto進行トースト）

## システム連携の前提
- Manus：字幕生成、テンプレワークフロー実行
- n8n：スライド→動画、Sora連携などノーコードパイプライン
- Discord：コミュニティ＆サポート拠点
- メルマガ/LINE：テストユーザー告知・オンボーディングの通知チャネル

## オペレーション要件
- メルマガ設計とnote導線をセットにした成長ループ
- アイキャッチ制作、ウェビナー運営、コンテンツ制作チームを想定
- テストユーザー30人（AIインフルエンサー20＋一般10）を対象に1on1ヒアリング
