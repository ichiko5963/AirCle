# Product Roadmap (v1→v3)

## v1：編集工程の自動化（基盤フェーズ）
| 項目 | 具体要件 |
| --- | --- |
| 字幕自動化 | AI文字起こし→字幕整形→SRT生成→Manus連携→誤字修正 |
| 無音カット完全自動化 | VAD発話判定、自然なカットライン、ノイズ除去、事故カット防止 |
| 字幕補正 | 句読点・フィラー・文脈修正、自動整形 |
| テロップ要約 v1.5 | 重要箇所の抽出と字幕反映 |

**目的：** 編集の地味作業をゼロにし、ノータッチ基盤を確立。

## v2：コンテンツ生成自動化（拡張フェーズ）
| 生成器 | 提供価値 |
| --- | --- |
| 切り抜き大量生成 | 1本から10案、TikTok/YTに最適化したプレビュー提案 |
| 要約動画生成 | 長尺→30秒ショート、ハイライト化＋スクリプト生成 |
| テーマ別ショート | 雑学、ニュース、スポーツハイライト、AIおばあちゃんなどをManus一発実行 |
| 商品UGC生成 | TikTok Shop向けUGCを完全自動生成、ショート広告最適化 |

**目的：** “制作=生成” を同列に扱い、投稿ネタ供給のボトルネックを解消。

## v3：動画制作エージェント（完成フェーズ）
- **完全自動ライン**：アップロード→カット/字幕/テロップ/要約/切り抜き/企画/サムネ/投稿文/SNS予約までエンドツーエンド
- **運用エージェント**：投稿時間最適化、コメント解析、再編集案、新規ショート自動生成、成長戦略提案
- **学習型プロデューサーAI**：伸びた動画を学習しユーザーごとの編集判断をチューニング
- **プラットフォーム統合**：YouTube/TikTok/Instagram/Xと連携し、予約投稿＆分析を一元化

**目的：** “動画運用を自動で回すOS”として、サブスク売上150万円/月を支える旗艦機能群。

## ロードマップのマイルストーン
1. **2025-11**：v1実装＋優先利用券プログラム開始（ノート経由で濃いユーザーを招待）
2. **2025-12**：テストユーザー30人にv1を提供し、v2の要件定義とUX改善のフィードバック収集
3. **2026 Q1**：切り抜き10案自動提案やニュース/雑学ジェネレーター（v2）を一般提供
4. **2026 Q2**：エージェント機能群のクローズドβ→v3ローンチ、投稿自動化までを提供
