### 1. プロダクト戦略の方向性

**【コンテンツ販売からプロダクト主軸への転換】**
- **市岡：** 当初、コンテンツ販売を検討していたが、プロダクトに主軸を置く方針に変更したい
  - 理由①：お金を払ってプロダクトに興味を持ってくれるユーザーを増やす
  - 理由②：開発資金のキャッシュフロー前倒し
  - 理由③：リスクブランディングを避ける（情報商材的なイメージを避けたい）
- **光る：** 賛成。プロダクトでいく方がいい
- **決定：** コンテンツ販売（有料ノート）は避け、プロダクトの買い切り版を作成してローンチする

**【買い切り版の設計】**
- 海外プロダクトでよく見られる「先行買い切り版」を採用
- 価格：半年分程度（18,000円前後を想定）
- 買い切りユーザーには基本的に無料で使わせる（ただし月額利用キャップは設ける）
- 設計上は約3年分の利用が可能

**【ノートコンテンツの扱い】**
- 有料販売は避けるが、無料公開やプレゼント企画での配布は検討
- 今までのまとめノートを作成し、無料で提供する方向

### 2. マーケティング戦略とファン層の設計

**【ユーザー層の分類】**
- 市岡がキャンバスで共有：Xのユーザーを以下の3層に分類
  1. マインド低めの層（一般層）
  2. 若干興味ある層（興味あり層）
  3. コアファン層

- サブスクは識別が高いため、最初は「興味あり層」と「コアファン層」が中心になると想定
- **目標：** コアファンをいかに増やすかが重要

**【オープンチャットとDiscordの使い分け】**
- **オープンチャット：** 
  - 広く浅く集客するための窓口
  - X投稿、プレゼント企画、ノート企画で集客
  - 公式LINE、メルマガからも誘導
  - 目標：3,000人規模（ストレッチ目標）
  
- **Discord：**
  - コアファン・購入者向けのクローズドコミュニティ
  - プロダクトの最新情報発信が中心
  - コミュニティ運営は市岡の得意分野ではないため、情報発信に特化
  - 購入者限定チャンネルを設ける

**【顧客育成フロー】**
```
X → オープンチャット → メルマガ → Discord（購入者）
イベント → オープンチャット → メルマガ → Discord（購入者）
公式LINE → オープンチャット → メルマガ → Discord（購入者）
```

- 市岡が公式LINE・オープンチャット・メルマガで顧客育成を担当
- 光るがXでバズらせ、新規ユーザーを流入させる
- イベントで入ってきた人もオープンチャットに誘導してリスト化
