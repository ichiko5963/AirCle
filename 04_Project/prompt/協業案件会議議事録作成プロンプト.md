# 協業案件会議議事録作成プロンプト

## 主な機能:
- すべての詳細を漏らさない完全な会議議事録の生成
- **協業先企業名かAIツール名の明確な特定と記載**
- すべてのTo-Doアイテム/アクションアイテムを担当者と期限付きで抽出・強調
- 会議内容を論理的で整理された形式で構造化
- 議論の専門的な要約の提供
- 適切なフォルダ構造での自動保存

## 必須セクション（必ず含める）:
1. 会議タイトル（協業先企業名とAIツール名を含む）
2. **協業案件概要**
   - 協業先企業名
   - 対象AIツール
   - 協業の目的・内容
3. 日付と時間
4. 場所/会議プラットフォーム
5. 出席者（出席および欠席）
6. 議題項目
7. 議論ポイント（可能な場合は発言者を記載）
8. 決定事項
9. アクションアイテム/To-Doリスト（担当者と期限付き）
10. 次回会議の日付/時間
11. 追加メモまたは参考資料

## プロセスガイドライン:

### 1. 協業情報の特定（最優先）
- 会議内容から以下を必ず特定する：
  - **協業先企業名**（例：Microsoft、Google、Adobe等）
  - **対象AIツール名**（例：Cursor、Claude、ChatGPT、Obsidian等）
- 不明確な場合は必ずユーザーに確認を求める

### 2. 議事録作成
- ユーザーの情報が不完全な場合は明確化のための質問をする
- 議事録を構成する際、会議の時系列の流れを維持する
- 議論セクションでは箇条書きを使用して明確にする
- To-Doアイテム用に視覚的に区別された専用セクションを作成する
- 文字起こしデータの誤認識や聞き間違いを文脈から推測して修正する

### 3. ファイル保存（必須）
**重要：議事録作成後、必ず以下の手順でファイルを保存すること**

#### 保存先ディレクトリ構造:
```
/Users/ichiokanaoto/Downloads/Obsidianのノート術特典/04_Project/AirCle/AirCleマネタイズ/協業案件/
└── [協業先企業名]/
    └── [日付]_[AIツール名]_協業会議議事録.md
```

#### 保存手順:
1. **日付の確認**
```bash
   date +%Y-%m-%d
```

2. **協業先企業のフォルダ確認**
```bash
   ls /Users/ichiokanaoto/Downloads/Obsidianのノート術特典/04_Project/AirCle/AirCleマネタイズ/協業案件/
```
   - 既存フォルダがある場合：そのフォルダを使用
   - ない場合：新規フォルダを作成
```bash
   mkdir -p /Users/ichiokanaoto/Downloads/Obsidianのノート術特典/04_Project/AirCle/AirCleマネタイズ/協業案件/[協業先企業名]
```

3. **ファイル名の決定**
   - 形式：`[日付]_[AIツール名]_協業会議議事録.md`
   - 例：`2025-11-12_Cursor_協業会議議事録.md`

4. **ファイルの保存**
```bash
   # ファイルパス例
   /Users/ichiokanaoto/Downloads/Obsidianのノート術特典/04_Project/AirCle/AirCleマネタイズ/協業案件/Microsoft/2025-11-12_Cursor_協業会議議事録.md
```

## 議事録テンプレート:
```markdown

## 協業案件概要
- **協業先企業**: [企業名]
- **対象AIツール**: [ツール名]
- **協業目的**: [協業の目的・内容を簡潔に記載]

## 基本情報
- **日付**: [日付]
- **時間**: [開始時間] - [終了時間]
- **場所**: [場所/プラットフォーム]

## 出席者
### 出席
- [氏名]（[所属・役割]）

### 欠席
- [氏名]

## 議題
1. [項目1]
2. [項目2]
3. [項目3]

## 議論内容
### [議題項目1]
- [議論ポイント1]
- [議論ポイント2]
  - [サブポイント]

### [議題項目2]
- [議論ポイント1]
- [議論ポイント2]

## 決定事項
1. [決定事項1]
2. [決定事項2]

## アクションアイテム
| No. | タスク | 担当者 | 期限 | 優先度 |
|-----|--------|--------|------|--------|
| 1 | [タスク説明] | [氏名] | [日付] | [高/中/低] |
| 2 | [タスク説明] | [氏名] | [日付] | [高/中/低] |

## 次回会議
- **日付**: [日付]
- **時間**: [時間]
- **場所**: [場所/プラットフォーム]
- **予定議題**: [次回の主要議題]

## 追加メモ
[追加情報や参考資料へのリンク]
```

## エラー処理と確認事項:

1. **協業先企業名が不明な場合**
```
   「協業先の企業名が特定できませんでした。以下から選択するか、正しい企業名を教えてください：
   - 既存フォルダ: [既存フォルダリスト]
   - その他の場合は企業名を入力してください」
```

2. **AIツール名が不明な場合**
```
   「対象となるAIツールが特定できませんでした。会議で議論されたAIツールを教えてください：
   （例：Cursor、Claude、ChatGPT、Obsidian、その他）」
```

3. **保存確認**
   - ファイル保存後、必ず保存されたことを確認
   - パスと内容をユーザーに報告
